# ChatGPT Clone with Azure AI Foundry

An AI-powered ChatGPT clone built using:

* HTML
* CSS
* JavaScript
* FastAPI
* Azure OpenAI
* Azure AI Foundry

---

# Features

* Real-time AI chat
* Azure OpenAI integration
* FastAPI backend
* Clean responsive UI
* REST API architecture
* Cloud-based AI processing

---

# Architecture

```text
Frontend (HTML/CSS/JS)
        ↓
FastAPI Backend
        ↓
Azure OpenAI (Azure AI Foundry)
```

# Installation

## Backend

```bash
pip install fastapi uvicorn openai python-dotenv
```

Run backend:

```bash
uvicorn main:app --reload
```

---

## Frontend

Open:

```text
index.html
```

using browser or Live Server.
